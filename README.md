## Tech Stack

## Run Locally


```bash
  cd  tracker
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Challenges
When page refresh the data is wiped out.So use local storage of the browser.Save the data and whenever the page is refresh the data remain same.
